# 3-kita
Тренировочный проект - главная страница сайта рекламного агенства.

Был выполнен редизайн существующего сайта (макет отрисовывался в Figma). 

В работе использовались следующие технологии:
1) HTML5
2) препроцессор SCSS
3) таск-менеджер Gulp
4) JavaScript
